# Assets folder — save your Unsplash images here

The playbill references **5 images** by filename. Save each from your Unsplash Robin Hood collection with the exact filenames below. The page will reference them as soon as they exist.

## Required images

| Filename | Which photo | Where it appears |
|---|---|---|
| `forest-gate.jpg` | **Stone castle gateway in mossy forest** | Main backdrop (fixed, fills the screen) |
| `arrows.jpg` | **Wall of feathered arrows** | Hero banner below the title |
| `coins.jpg` | **Pile of golden coins** | Texture behind the ticket prices |
| `campfire.jpg` | **Campfire in forest** | Accent before the welcome note |
| `sunset.jpg` | **Medieval abbey at sunset / forest valley** | Finale image near the licensing footer |

## How to save

1. In the chat / on Unsplash, right-click each image → **Save image as...**
2. Rename using the exact filenames above (all lowercase, `.jpg` extension)
3. Place in this folder: `sherwood-deploy/assets/`

## If a file is missing

The page still works. Each accent band falls back to a dark background with no image — the playbill reads fine either way.
