# Sherwood! A Merry Playbill

The comical sibling of `sherwood-playbill/`. Same show, different mood. Sunny
daylight greenwood, bouncy SVG icons, playful copy, no torch darkness.

Built with HTML, CSS, and vanilla JavaScript. Audio via Howler.js.

## Run it locally

```bash
python -m http.server 8000
```

Then visit http://localhost:8000

## Structure

```
sherwood-playbill-comical/
  index.html          entry, playful copy
  css/style.css       bright palette, Henny Penny + Caveat fonts
  js/audio.js         ambient Celtic loop (Howler)
  js/scene.js         gate, parallax, hotspots, flavor card, crossfade
  assets/
    audio/celtic-impulse.mp3
    img/              sunny-greenwood, merry-tavern, daylight-castle,
                      feast, archer, pennants (Pexels, CC0)
```
